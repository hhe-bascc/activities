Keeping this page current

This page is a single file. Everything on it is edited in one list near the bottom of the code — you don't have to touch the layout or the styling to add a resource.

How to turn on a "Coming soon" tile
  1. Open this file in any text editor (Notepad, TextEdit, VS Code).
  2. Search for the tile's title — for example Parent Handbook.
  3. On that line, replace url:null with the real link in quotes, like url:"https://…".
  4. Save. The tile turns live automatically — the badge, the hover, and the link all follow from whether a URL is present.
// before
{t:"Parent Handbook", i:"i-family", c:"plum", url:null, d:"…"}

// after
{t:"Parent Handbook", i:"i-family", c:"plum", url:"https://drive.google.com/…", d:"…"}
Adding a resource that isn't listed

Copy any existing line, paste it below, and change the four fields. Put it inside whichever section it belongs to.

{t:"Title of the resource",
 i:"i-clipboard",          // icon name — see the list at the top of the file
 c:"royal",                // color: navy, royal, grass, coral, plum, chalk
 url:"https://…",          // or null if it doesn't exist yet
 d:"One line about what it's for."}
Where the files need to live

The two activity guides are linked by filename, not by web address. That means this page and both guides have to sit in the same folder — the same Drive folder, the same shared drive, or the same web folder if it gets hosted.

Expected filenames:

  1. index.html — this page
  2. HHE-BASCC-Indoor-Activity-Guide.html
  3. HHE-BASCC-Outdoor-Activity-Guide.html

If a guide moves somewhere with its own web address, swap the filename for that full address the same way you would for any other tile.

Getting this onto counselors' phones

Once the folder is somewhere counselors can reach, send them the link to this page and have them add it to their home screen:

  1. iPhone — open the link in Safari, tap Share, then Add to Home Screen.
  2. Android — open in Chrome, tap the three-dot menu, then Add to Home screen.

It then opens like an app, which matters when it's 3:15 and it's raining.
